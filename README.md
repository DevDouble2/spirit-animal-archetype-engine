## 🛠️ Technical Overview
This repository holds the interactive logic for the **iSpiritAnimalQuiz**. Moving past static forms, the engine emphasizes:

* **Archetype Vector Mapping**: Correlating situational user responses with 20+ distinct psychological models represented as animal profiles.
* **Zero-Retention Privacy**: 100% of the diagnostic logic is executed in the user's browser (client-side), ensuring no personal data is transmitted or stored.
* **Stochastic Scoring**: Advanced scoring matrices where each choice influences multiple personality dimensions (Dominance, Strategy, Social, etc.).

### Key Features
- **Client-Side Privacy**: High-performance JavaScript execution ensuring complete anonymity for the user.
- **n-Dimensional Logic**: Calculates the "Euclidean distance" between user responses and animal archetype vectors for maximum accuracy.
- **Multi-Language Support**: Fully localized for global audiences, including English, Spanish, French, Japanese, Korean, and Chinese.

---

## 📊 Core Methodology
1. **Behavioral Input Layer**: Captures 60+ data points across intuitive, situational decision-making scenarios.
2. **Vector Analysis Layer**: Normalizes scores against an internal database of animal behavioral archetypes.
3. **Synthesis Layer**: Dynamically renders a comprehensive "Spirit Animal Profile" with actionable personal insights.
